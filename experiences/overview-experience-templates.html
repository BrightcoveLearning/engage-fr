Généralités sur les modèles d'expérience

Dans cette rubrique, vous découvrirez les différents modèles d'expérience disponibles.

Les modèles d'expérience contrôlent l'aspect général de l'expérience. Lors de la création d'une nouvelle expérience, vous sélectionnerez un modèle à utiliser pour l'expérience. Notez qu'après avoir sélectionné un modèle, le modèle ne peut pas être modifié.

Les modèles suivants sont disponibles.

  • Application mobile
  • Modèle Web

Chaque modèle fournit un ensemble de modèles de page qui peuvent être personnalisés et/ou prévisualisés dans l'Éditeur d'expérience. Les modèles de page peuvent être visualisés en éditant une expérience puis en cliquant sur PAGES dans le menu de navigation de gauche.

Cliquez sur un nom de page pour ouvrir la mise en page dans l'éditeur. Dans l'exemple ci-dessus, Accueil , Les collections , Détails de la vidéo , Chercher et Profil des pages sont disponibles.

Modèle d'application mobile

Le modèle d'application mobile est utilisé pour créer une application mobile native. Une fois publié, il est accessible sur les appareils iOS et Android en utilisant le Brightcove s'engager™ application.

Le modèle d'application mobile fournit les modèles de mise en page suivants :

  • Accueil
  • Collections
  • Détails de la vidéo
  • Rechercher
  • Profil

Modèle Web

Le modèle Web comporte une page d'accueil centrée sur la vidéo et orientée image. Le modèle Web prend en charge la vidéo facilement détectable, survolez les descriptions vidéo et les catégories et sous-catégories de contenu.

Le modèle Web fournit trois modèles de mise en page.

  • Accueil
  • Catégorie
  • Détails de la vidéo
