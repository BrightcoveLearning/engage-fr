Utilisation de l'Editeur d'expérience pour personnaliser une expérience Web

Dans cette rubrique, vous apprendrez comment utiliser l'Éditeur d'expérience pour personnaliser l'apparence d'une expérience Web.

L'éditeur d'expérience est utilisé pour personnaliser l'apparence et la convivialité des expériences créées à l'aide de Brightcove Engage™. Cliquez sur le titre de l'expérience pour ouvrir l'éditeur d'expérience. Avec Web Experiences, l'Éditeur d'expérience offre la possibilité de :

  • Ajouter des objets aux pages
  • Ajouter des vidéos à un site
  • Personnaliser le style du site
  • Contrôler l'accès au site
  • Configurer les détails du site

Ces options sont accessibles en cliquant sur les menus de la navigation de gauche.

Ajout d'objets à une page

Des objets tels que du texte, des images et des URL peuvent être ajoutés aux pages, ce qui vous permet de personnaliser et de marquer votre expérience Web. Pour savoir comment ajouter des objets aux pages, voir Ajout d'images, de texte et de liens aux pages d'expérience Web.

Ajout de vidéos à une expérience Web

Pour ajouter des vidéos à une expérience, cliquez sur VIDÉOS dans la navigation de gauche. Pour plus d'informations sur l'ajout de vidéos à une expérience Web, consultez Ajout de vidéos à une expérience.

Personnalisation du style

Pour personnaliser les couleurs et les polices utilisées dans une expérience Web ou pour sélectionner un autre thème, cliquez sur STYLE dans la navigation de gauche. Pour plus d'informations sur la personnalisation du style, voir Personnalisation du style d'une expérience Web.

Contrôle de l'accès au site

Les paramètres de contrôle d'accès sont utilisés pour restreindre l'accès au site. Pour configurer les paramètres de contrôle d'accès, cliquez sur CONTRÔLE D'ACCÈS dans la navigation de gauche.

Pour plus d'informations sur les paramètres de contrôle d'accès, consultez Contrôle de l'accès à une expérience.

Configuration des détails de l'expérience

Les paramètres Vue d'ensemble permettent de modifier le nom, la description et la langue de l'expérience. Pour configurer les détails de l'expérience, cliquez sur APERÇU dans la navigation de gauche.

Pour plus de détails sur la configuration des détails du site, voir Modification des détails de l'expérience.
