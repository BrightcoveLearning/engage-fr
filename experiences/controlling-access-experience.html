Contrôle de l'accès à une expérience

Dans cette rubrique, vous apprendrez comment restreindre l'accès à une expérience à l'aide de profils de contrôle d'accès.

Les profils de contrôle d'accès sont utilisés pour contrôler l'accès aux expériences et sont créés dans le cadre des paramètres du projet. Une fois les profils créés, ils peuvent être affectés à des expériences pour contrôler l'accès. Pour plus d'informations sur la création de profils de contrôle d'accès, voir Création de profils de contrôle d'accès.

Pour affecter un profil de contrôle d'accès à une expérience, procédez comme suit :

  1. Modifiez l'expérience.
  2. Cliquez sur CONFIGURATION DU SITE> Contrôle d'accès dans la navigation de gauche.
  3. Sélectionnez un profil de contrôle d'accès pour l'expérience.
  4. Cliquez sur Save.

Assurez-vous de republier l'expérience après avoir apporté des modifications aux paramètres de contrôle d'accès.
