Modification des détails de l'expérience

Dans cette rubrique, vous apprendrez comment modifier les détails d'une expérience.

Le titre, la description et la langue de l'expérience peuvent être configurés dans le cadre des paramètres Vue d'ensemble. Pour configurer les paramètres Vue d'ensemble, modifiez l'expérience, puis cliquez sur APERÇU dans la navigation de gauche.

La page Vue d'ensemble s'affiche :

  • Type d'expérience - Affiche le type d'expérience
  • Titre et description de l'expérience - Lorsque vous créez une nouvelle expérience pour la première fois, vous êtes invité à saisir le Titre de l'expérience et Description de l'expérience ; Ces valeurs peuvent être modifiées ici
  • Expérience de la langue - Définit le texte spécifique de l'expérience dans cette langue (Expériences Web uniquement)
  • Politique de confidentialité - Entrez l'URL de votre politique de confidentialité (application mobile uniquement)
  • Termes et conditions - Saisissez l'URL de vos conditions générales (application mobile uniquement)
  • E-mail du contact d'aide - Entrez l'adresse e-mail à utiliser si les utilisateurs ont des problèmes / questions avec l'utilisation de l'application (application mobile uniquement)

Lorsque vous avez terminé, cliquez sur Enregistrer pour enregistrer vos modifications. Notez que les liens vers la politique de confidentialité, les conditions générales et l'e-mail de contact d'aide apparaîtront sur la page Profil de l'application mobile.

Remarque : Après avoir apporté des modifications à une expérience, assurez-vous de republier l'expérience.
