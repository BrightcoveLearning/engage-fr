Utilisation de l'éditeur d'expérience pour personnaliser une expérience d'application mobile

Dans cette rubrique, vous apprendrez comment utiliser l'Éditeur d'expérience pour personnaliser l'apparence d'une expérience d'application mobile.

L'éditeur d'expérience est utilisé pour personnaliser l'apparence et la convivialité des expériences créées à l'aide de Brightcove Engage™. Cliquez sur le titre de l'expérience pour ouvrir l'éditeur d'expérience. Avec les expériences d'applications mobiles, l'éditeur d'expérience offre la possibilité de :

  • Ajouter un logo à l'application
  • Ajouter des vidéos à une application
  • Personnalisez le style de l'application
  • Configurer le contrôle d'accès
  • Configurer les détails de l'application

Ces options sont accessibles en cliquant sur les menus de la navigation de gauche.

Ajout d'un logo à l'application

Un logo peut être ajouté en haut d'une expérience d'application mobile vous permettant de marquer l'expérience. Pour savoir comment ajouter un logo, voir Ajout d'un logo à une expérience d'application mobile.

Ajout de vidéos à une application

Pour ajouter des vidéos à une expérience, cliquez sur VIDÉOS dans la barre de navigation de gauche. Pour savoir comment ajouter des vidéos à une expérience, consultez Ajout de vidéos à une expérience.

Personnalisation du style de l'application

Pour personnaliser les couleurs et le thème utilisés dans une expérience d'application mobile, cliquez sur STYLE dans la navigation de gauche. Pour plus d'informations sur la personnalisation du style, voir Styliser les expériences des applications mobiles.

Configuration du contrôle d'accès

Les paramètres de contrôle d'accès permettent de restreindre l'accès à l'application. Les utilisateurs de l'application devront s'authentifier lorsqu'ils ouvrent l'application. Pour configurer les paramètres de contrôle d'accès, cliquez sur CONTRÔLE D'ACCÈS dans la barre de navigation de gauche.

Pour plus d'informations sur les paramètres de contrôle d'accès, consultez Contrôle de l'accès à une expérience.

Configuration des détails de l'application

Les paramètres Vue d'ensemble permettent de modifier le nom, la description et la langue d'une expérience. Pour configurer les détails de l'expérience, cliquez sur APERÇU dans la navigation de gauche.

Pour plus d'informations sur la configuration des détails de l'expérience, consultez Modification des détails de l'expérience.
