Configuration des paramètres d'expérience

Dans cette rubrique, vous apprendrez comment configurer les paramètres d'expérience.

Pour configurer les paramètres d'expérience, ouvrez le Projets et cliquez sur le Réglages lien en haut de la page. Les paramètres suivants peuvent être configurés :

  • Profils de contrôle d'accès

Configuration des profils de contrôle d'accès

Il est possible de créer des profils de contrôle d'accès pour contrôler l'accès aux expériences publiées. L'accès à un site peut être limité par :

  • Informations d'identification SSO - Un nom d'utilisateur dont un mot de passe est requis pour l'accès

Pour plus d'informations sur la création de profils de contrôle d'accès, voir Création de profils de contrôle d'accès.
