Sécurisation des expériences à l'aide de l'authentification sans mot de passe Auth0

Dans cette rubrique, vous apprendrez comment sécuriser les expériences à l'aide de l'authentification Auth0 Passwordless.

Les expériences peuvent être sécurisées en utilisant Auth0 avec authentification sans mot de passe. Lors de l'utilisation de cette méthode de sécurité, les utilisateurs sont invités à entrer un numéro de téléphone. Auth0 envoie un code à usage unique en utilisant Twilio. L'utilisateur entre ensuite le code dans votre application. Si le numéro de téléphone associé au code correspond à un utilisateur existant, Auth0 authentifie l'utilisateur.

Remarque : Les étapes et les images de cette rubrique utilisent des versions d'essai de Auth0 et Twilio.

Préliminaires

Avant de configurer Auth0 Passwordless, effectuez les tâches suivantes :

  • Configurez votre expérience d'application mobile en utilisant Auth0 comme indiqué dans Sécuriser les expériences avec Auth0
  • Inscrivez-vous à un Twilio compte d'essai et vérifiez votre compte

Obtenir un numéro de téléphone Twilio

Vous devez obtenir un numéro de téléphone Twilio qui sera utilisé comme numéro d'expéditeur lorsque vous configurez sans mot de passe.

  1. Connectez-vous à votre compte Twilio.
  2. Sur le Tableau de bord page, cliquez sur Obtenez un numéro d'essai.
  3. Cliquez sur Choisissez ce numéro.
  4. Cliquez sur Terminé pour confirmer le numéro.

Configuration d'une connexion sans mot de passe dans Auth0

Commencez par configurer une connexion sans mot de passe dans Auth0.

  1. Connectez-vous à votre compte Auth0.
  2. Dans la navigation de gauche, cliquez sur Connexions> Sans mot de passe.
  3. Activer le SMS connexion.

Saisir votre Twilio SID

  1. Revenir à la page Twilio.
  2. Sur la page Tableau de bord, recherchez le COMPTE SID champ.
  3. Copiez la valeur dans le presse-papiers.
  4. Revenir à Auth0.
  5. Collez la valeur dans le Twilio SID champ.

Saisir votre jeton d'authentification Twilio

  1. Retournez à Twilio.
  2. Sur la page Tableau de bord, recherchez le JETON AUTH champ.
  3. Clique le Spectacle lien pour révéler le jeton.
  4. Copiez la valeur dans le Presse-papiers.
  5. Revenir à Auth0.
  6. Collez la valeur dans le Jeton d'authentification Twilio champ.

Saisir votre numéro de téléphone Twilio

  1. Retournez à Twilio.
  2. Sur la page Tableau de bord, recherchez le NUMÉRO D'ESSAI toi créé plus tôt.
  3. Copiez le numéro dans le Presse-papiers.
  4. Revenir à Auth0.
  5. Collez la valeur dans le De champ.
  6. (Facultatif) Personnalisez le Message texte.

Activation de votre application

  1. Clique le Applications lien en haut de la boîte de dialogue.
  2. Activez l'application Auth0 que vous avez créée pour sécuriser votre application Brightcove Engage.
  3. Cliquez sur sauver et fermez la boîte de dialogue.

Configuration de votre application pour utiliser une connexion sans mot de passe

  1. Dans la navigation de gauche, cliquez sur Connexion universelle.
  2. Clique le S'identifier lien.
  3. Activer le Personnaliser la page de connexion option.
  4. Sélectionner Verrouiller (sans mot de passe) du MODÈLES PAR DÉFAUT la liste déroulante.
  5. Cliquez sur SAUVEGARDER LES MODIFICATIONS.

Aperçu de l'expérience

Une fois les étapes ci-dessus terminées, vous pouvez prévisualiser votre expérience pour confirmer que le spectateur est invité à entrer un numéro de téléphone.

  1. Connectez-vous à Brightcove Engage.
  2. Cliquez sur le titre de l'expérience qui a été sécurisé avec Auth0.
  3. Prévisualisez l'expérience. Vous devriez avoir un écran de connexion qui vous invite à entrer un numéro de téléphone.
  4. Fermez l'aperçu.

 

Ajout de numéros de téléphone valides à Twilio

Ajoutez un ou plusieurs numéros de téléphone à Twilio à des fins de test.

  1. Retournez à Twilio.
  2. Dans la navigation de gauche, cliquez sur le Tous les produits et services icône.
  3. Dans la navigation de gauche, cliquez sur Les numéros de téléphone.
  4. Dans la navigation de gauche, cliquez sur Identifiants d'appel vérifiés.
  5. Entrez les numéros que vous souhaitez utiliser pour les tests. Votre compte d'essai doit avoir le numéro de téléphone utilisé lors de la création de votre compte d'essai.

Validation de la configuration

La dernière étape consiste à tester que les spectateurs peuvent se connecter à l'aide de leur numéro de téléphone mobile.

  1. Ouvrez l'application Brightcove Engage sur votre appareil. Si vous y êtes invité, entrez le Code d'accès pour votre expérience.
  2. Entrez votre numéro de téléphone et cliquez sur SOUMETTRE.
  3. Confirmez que vous recevez un message SMS avec un code de vérification.
  4. Entrez le code de vérification et cliquez sur SOUMETTRE.

Votre application mobile Brightcove Engage doit s'afficher.
