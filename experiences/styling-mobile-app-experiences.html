Styliser les expériences des applications mobiles

Dans cette rubrique, vous apprendrez comment styliser une expérience d'application mobile.

À l'aide de l'Éditeur d'expérience, vous pouvez personnaliser le style d'une expérience d'application mobile en modifiant le thème et les couleurs. Pour modifier le style, modifiez l'expérience, puis cliquez sur STYLE dans la navigation de gauche.

Sélection d'un thème

Sélection d'un Thème applique un jeu de couleurs prédéfini à l'expérience. Pour changer le Thème:

  1. Cliquez sur STYLE dans la barre de navigation de gauche.
  2. Sélectionnez un thème dans la liste déroulante. Au fur et à mesure que vous sélectionnez un thème, l'aperçu est actualisé.
  3. Pour revenir à l'éditeur d'expérience, cliquez sur < Retour à l'expérience.

Changement de couleurs

Les Couleurs Les paramètres peuvent être utilisés pour affiner les couleurs fournies par un thème. Par exemple, les paramètres de couleurs permettent de personnaliser les couleurs d'arrière-plan et de police. Les couleurs suivantes peuvent être changées :

  • Accent - Contrôle la couleur d'arrière-plan lors du chargement de l'application, la couleur de la barre en haut de l'application et la couleur de la police pour les noms de collection sur la page d'accueil
  • Texte principal - Contrôle la couleur de la police du Voir tout boutons et titres vidéo qui apparaissent sous les images d'affiche vidéo sur la page d'accueil
  • Texte secondaire - Contrôle la couleur des titres qui apparaissent sur les pages Collections, Détails de la vidéo et Profil
  • Icônes - Contrôle la couleur des icônes de navigation, de recherche et d'utilisateur en haut de l'application
  • Contexte - Contrôle la couleur d'arrière-plan de l'application

Pour personnaliser les couleurs:

  1. Cliquez sur STYLE icône dans la navigation de gauche. le Couleurs La section affichera tous les objets disponibles et leurs couleurs.
  2. Pour sélectionner une nouvelle couleur, cliquez sur un cercle de couleur.
  3. En utilisant le sélecteur de couleur, sélectionnez une couleur en cliquant sur celle-ci dans le carré de couleur ou en saisissant un numéro de couleur hexadécimal au-dessous du carré de couleur. Notez que vous pouvez modifier la plage de couleurs en cliquant sur le spectre de couleurs sous le carré de couleur. Cliquez sur X pour fermer le sélecteur de couleurs et enregistrer la modification.

    Pour réinitialiser une sélection de couleurs, cliquez sur la flèche vers la gauche du cercle de couleurs.

  4. Pour revenir à l'Éditeur d'expérience, cliquez sur < Retour à l'expérience.
