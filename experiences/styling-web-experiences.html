Personnaliser le style d'une expérience Web

Dans cette rubrique, vous apprendrez comment personnaliser le style d'une expérience Web.

À l'aide de l'Éditeur d'expérience, vous pouvez personnaliser le style d'une expérience Web en modifiant le thème, les couleurs et les polices. Pour modifier le style, modifiez l'expérience, puis cliquez sur STYLE dans la navigation de gauche.

Sélection d'un thème

Sélection d'un Thème applique un ensemble prédéfini de polices et de couleurs à l'expérience. Pour sélectionner un Thème:

  1. Cliquez sur STYLE dans la navigation de gauche.
  2. Sélectionnez un thème dans la liste déroulante. Au fur et à mesure que vous sélectionnez un thème, l'expérience s'actualise afin que vous puissiez prévisualiser le thème.
  3. Pour revenir à l'éditeur d'expérience, cliquez sur < Retour à l'expérience.

Personnalisation des couleurs

Les paramètres Couleurs peuvent être utilisés pour affiner les couleurs fournies par un thème. Par exemple, les paramètres de couleurs permettent de personnaliser les couleurs d'arrière-plan et de police. Pour personnaliser les couleurs:

  1. Cliquez sur l'icône STYLE dans la navigation de gauche. La section Couleurs affiche tous les objets de page et leurs couleurs.
  2. Pour sélectionner une nouvelle couleur, cliquez sur le cercle de couleur.
  3. En utilisant le sélecteur de couleur, sélectionnez une couleur en cliquant sur celle-ci dans le carré de couleur ou en saisissant un numéro de couleur hexadécimal au-dessous du carré de couleur. Notez que vous pouvez modifier la plage de couleurs en cliquant sur le spectre de couleurs sous le carré de couleur. Cliquez sur X pour fermer le sélecteur de couleurs et enregistrer la modification.

    Pour réinitialiser une sélection de couleurs, cliquez sur la flèche vers la gauche du cercle de couleurs.

  4. Pour revenir à l'Éditeur d'expérience, cliquez sur < Retour à l'expérience.

Sélection des polices

Pour sélectionner différentes polices pour l'expérience :

  1. Cliquez sur STYLE dans la navigation de gauche.
  2. Sélectionnez les nouvelles polices à l'aide des listes déroulantes.

    Pour réinitialiser une sélection de police, cliquez sur le lien de réinitialisation au-dessus de la liste déroulante de police.

  3. Pour revenir à l'Éditeur d'expérience, cliquez sur < Retour à l'expérience.
