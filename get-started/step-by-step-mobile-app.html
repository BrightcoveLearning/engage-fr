Étape par étape : Créer et publier une expérience d'application mobile

Cette rubrique fournit un ensemble d'étapes que vous pouvez suivre pour créer et publier une expérience d'application mobile à l'aide de Brightcove Engage™.

Après avoir terminé ce sujet, vous devriez être en mesure de :

  • Utiliser Brightcove Engage pour créer une nouvelle expérience d'application mobile
  • Ajouter des vidéos à une expérience
  • Personnalisez une expérience
  • Prévisualisez et publiez une expérience
  • Installez l'application Brightcove Engage sur un appareil et accédez à une expérience
Remarque : Pour obtenir un guide pas à pas sur la création d'une expérience Web, consultez Pas à pas: Créer et publier une expérience Web.

Audience

Toute personne qui a besoin de créer des applications mobiles natives pour afficher du contenu vidéo d'entreprise.

Conditions préalables

  • Accès à Brightcove Engage (inscrivez-vous pour un essai)

Préliminaires

Ce guide étape par étape utilisera les vidéos incluses dans le compte d'essai Brightcove Engage. N'hésitez pas à télécharger et à utiliser vos propres vidéos.

Installation de l'application Brightcove Engage

L'application mobile Brightcove Engage est nécessaire pour afficher les expériences d'applications mobiles publiées. Suivez ces étapes pour installer l'application.

  1. Ouvrez le Magasin d'applications (iOS) ou le Google Play Store (Android) sur votre appareil.
  2. Rechercher le Brightcove s'engager application.
  3. Téléchargez et installez l'application sur votre appareil.

L'application sera utilisée plus tard pour voir l'expérience que nous allons créer.

Création d'une expérience d'application mobile

Pour créer une expérience d'application mobile à l'aide de Brightcove Engage, vous effectuerez les tâches suivantes :

  • Organiser des vidéos à l'aide de listes
  • Créez une nouvelle expérience
  • Modifier les détails du site
  • Ajouter un logo à la page principale
  • Créer des collections et ajouter des vidéos aux collections
  • Aperçu de l'expérience
  • Personnaliser le style
  • Publier l'expérience
  • Accédez à l'expérience sur un appareil

Organisation de vidéos à l'aide de playlists

Avant de commencer à créer des expériences, il est utile de réfléchir à la façon dont vous souhaitez organiser/regrouper vos vidéos. En règle générale, vous devez regrouper des vidéos associées. Par exemple, vous pouvez regrouper les vidéos du PDG, les vidéos RH ensemble, les vidéos marketing, etc. Lorsque les internautes utilisent votre expérience d'application mobile, ils verront une liste de ces groupes, puis choisissent les vidéos qu'ils souhaitent regarder.

En outre, lors de la création d'une expérience d'application mobile, vous devez sélectionner une seule vidéo pour être la « vidéo en vedette ». Cette vidéo sera affichée en vue en haut de l'application. Lorsque vous utilisez Brightcove Engage, les vidéos associées sont regroupées à l'aide de listes de lecture.

Remarque : Si vous utilisez vos propres vidéos, créez plusieurs listes de lecture pour organiser les vidéos.

Pour organiser les exemples de vidéos en listes de lecture, procédez comme suit.

  1. Connectez-vous à Brightcove Engage. Les Projets page apparaîtra.
  2. Clique le Vidéos onglet dans l'en-tête de navigation. Une liste des exemples de vidéos apparaîtra.
  3. Dans la barre de navigation de gauche, cliquez sur + Créer une liste de lecture.
  4. Définissez Playlist type sur Manual.
  5. Définissez Playlist Name sur Featured Video.
  6. Cochez Afficher la liste de lecture dans la barre latérale.
  7. Cliquez sur Save.
  8. En utilisant les mêmes paramètres, créez deux autres listes de lecture Sales Videos et HR Videos. Trois listes de lecture devraient être créées.
  9. Dans la barre de navigation de gauche, cliquez sur Toutes les vidéos.
  10. Cliquez sur la case à côté de Réunion de toutes les mains 2019 vidéo, puis cliquez sur Suite> Ajouter à la playlist.
  11. Sélectionnez le Vidéo en vedette liste de lecture, puis cliquez sur Ajouter.
  12. Sélectionnez les vidéos suivantes et ajoutez-les à la Vidéos de vente playlist.
    • Réunion de stratégie commerciale 2018
    • Guide de vente - Développer votre pipeline sur de nouveaux marchés - Ep 02
    • Guide de vente - Nouvelles techniques pour les excursions de vente incitative - Ep 01
  13. Sélectionnez les vidéos suivantes et ajoutez-les à la Vidéos RH playlist.
    • HR - Construire votre plan de communications internes - E02
    • HR - Intégration de nouveaux employés - E01

Toutes les vidéos sont désormais organisées en playlists. Cliquez sur un nom de liste de lecture dans la navigation de gauche pour afficher les vidéos de cette liste de lecture.

Pour plus d'informations sur les listes de lecture, consultez le Créer et gérer des listes de lecture sujet.

Créer une nouvelle expérience

Pour créer une expérience d'application mobile, procédez comme suit.

  1. Dans l'en-tête de navigation, cliquez sur le Projets languette.
  2. Confirmer le Créer de l'expérience page apparaît. Si une liste d'expériences apparaît, cliquez sur Nouvelle expérience pour afficher le Créer de l'expérience page.
  3. Localisez le Application mobile modèle et cliquez sur Choisir.
    Remarque : Une fois que vous avez sélectionné un modèle, il ne peut pas être modifié. Pour plus d'informations sur les modèles, consultez le Présentation des modèles d'expérience sujet.
  4. Entrer Exemple d'application mobile pour le Titre de l'expérience. Pour l'expérience Pre-Peupler ?, sélectionnez Non merci.
  5. Cliquez sur Créer une expérience.

L'éditeur d'expérience s'ouvrira. L'Éditeur d'expérience permet de personnaliser l'expérience et permet d'ajouter des vidéos, de choisir des styles et de configurer d'autres paramètres. Tous les paramètres de personnalisation sont accessibles à l'aide du menu de navigation de gauche.

Modification des détails du site

Le titre de l'expérience, la description et les paramètres de langue peuvent être modifiés à l'aide de l'éditeur d'expérience.

  1. Cliquez sur APERÇU dans la barre de navigation de gauche.
  2. Pour le Description de l'expérience , ajouter Exemple d'expérience d'application mobile.
  3. Cliquez sur Save.

Ajout d'un logo à la page principale

Chaque expérience est composée d'un ensemble de modèles de page. Les modèles de page fournis dépendent du modèle sélectionné lors de la création de l'expérience. Le modèle d'application mobile fournit Accueil , Les collections , Détails de la vidéo , Chercher et Profil modèles de page. Ces pages peuvent être vues dans la marge de gauche dans le cadre de la PAGE menu.

Le modèle d'application mobile permet d'ajouter un logo à la page d'accueil. Pour plus d'informations sur l'ajout d'un logo, voir Ajout d'un logo à une expérience d'application mobile.

Pour ajouter un logo à la page d'accueil, procédez comme suit.

  1. Cliquez sur PAGES > Accueil dans la barre de navigation de gauche.
  2. Clique le Ajouter un logo lien sur l'aperçu de l'expérience.
  3. Cliquez sur Feuilleter puis parcourez et sélectionnez une image à utiliser dans l'en-tête.
    Remarque : Brightcove redimensionnera l'image à 65 x 30 px.
    Remarque : Brightcove recommande d'utiliser des images d'une taille de < 20 Ko.
  4. Cliquez sur Save.
  5. Vérifiez que l'image apparaît dans l'aperçu de l'expérience.

Création de collections et ajout de vidéos aux collections

À l'aide de l'éditeur d'expérience, vous pouvez sélectionner les vidéos à afficher dans l'expérience. Lors de la création d'expériences, les vidéos sont organisées en collections. Lors de la création d'expériences d'applications mobiles, la première vidéo de la première collection est utilisée comme vidéo en vedette. La vidéo en vedette apparaît en haut de l'application.

Lors de la création de collections pour les expériences d'applications mobiles, Brightcove recommande de créer un Vidéo en vedette collection comme première collection. Nous attribuerons ensuite la liste de lecture vidéo en vedette que nous avons créée précédemment à cette collection.

Pour plus d'informations sur la création de collections et l'ajout de vidéos à une expérience, voir Ajout de vidéos à une expérience.

Pour ajouter des vidéos à l'expérience, procédez comme suit.

  1. Cliquez sur VIDÉOS dans la barre de navigation de gauche.
  2. Nommez la première collection Vidéo en vedette puis appuyez sur entrer.
  3. Cliquez sur le nom de la collection pour ouvrir l'éditeur de collection.
  4. Sélectionnez Playlist pour les critères.
  5. Clique le Vidéo en vedette playlist. La vidéo de cette liste de lecture apparaît dans la liste des vidéos. Cliquez sur Save.
    Remarque : Les Vidéo en vedette liste de lecture a été créée dans le Organiser des vidéos à l'aide de listes de lecture section.
    Remarque : Lors de l'utilisation d'une attribution de vidéo de liste de lecture, l'ordre des vidéos est basé sur les paramètres de liste de lecture qui ont été définis dans le Vidéos page.
  6. Créez une autre collection nommée Ventes en utilisant le Vidéos de vente playlist comme critère.
  7. Créer une autre collection nommée HEURE en utilisant le Vidéos RH playlist comme critère.
  8. Confirmez que 3 collections ont été créées.
    Remarque : Vous pouvez modifier l'ordre des collections en cliquant sur la collection et en la faisant glisser vers une nouvelle position.

Aperçu de l'expérience

La prévisualisation d'une expérience d'application mobile vous permet de voir comment elle s'affichera sur les appareils mobiles. Un aperçu rapide peut être fait en cliquant sur les modèles de page qui apparaissent sous le PAGES élément de navigation. Notez que lorsque vous utilisez cette option d'aperçu, les vidéos ne seront pas lues, mais vous pouvez obtenir une vue de la mise en page.

L'expérience peut également être prévisualisée dans un nouvel onglet de navigateur ou un lien partageable peut être généré pour que d'autres personnes puissent prévisualiser. Pour plus d'informations sur la prévisualisation et la publication d'une expérience d'application mobile, consultez Aperçu et publication d'une expérience d'application mobile.

Pour prévisualiser l'expérience, procédez comme suit.

  1. Lors de la modification du site à l'aide de l'Editeur d'expérience, cliquez sur Aperçu puis cliquez sur Aperçu réactif.
  2. L'aperçu s'ouvre dans un nouvel onglet de navigateur.
  3. Fermez l'onglet Aperçu du site.

Personnalisation du style

Les paramètres Style peuvent être utilisés pour personnaliser les couleurs utilisées dans l'expérience. Pour plus d'informations sur la modification du style des expériences d'application mobile, consultez Styliser les expériences des applications mobiles.

Pour modifier le style, procédez comme suit.

  1. Cliquez sur STYLE dans la barre de navigation de gauche.
  2. UNE Thème est un ensemble de couleurs qui peut être appliqué à un modèle. Sélectionnez le Lumière thème.
  3. Pour modifier une couleur, cliquez sur le cercle de couleur associé.
  4. Utilisez le curseur situé sous le carré de couleur pour sélectionner une plage de couleurs.
  5. Cliquez sur le carré pour sélectionner la nuance de la couleur voulue. La couleur peut également être définie en entrant le code hexadécimal approprié, la valeur RVB ou la valeur HSV (teinte, saturation, légèreté) dans le champ texte.
  6. Cliquez sur le lien < Retour à l'expérience en haut du menu Style.

Pour réinitialiser la couleur à sa valeur d'origine, cliquez sur la flèche gauche en regard du cercle de couleur.

Publication de l'expérience

Les expériences d'applications mobiles doivent être publiées avant d'être accessibles sur les appareils mobiles. Pour plus d'informations sur la prévisualisation et la publication d'une expérience d'application mobile, consultez Aperçu et publication d'une expérience d'application mobile.

Pour publier l'expérience, procédez comme suit.

  1. Dans l'éditeur d'expérience, cliquez sur Publier.
  2. Dans la boîte de dialogue Publier cette application mobile, cliquez sur Publier maintenant pour confirmer la publication.

La clé d'accès sera nécessaire pour accéder à l'expérience sur les appareils mobiles.

Accéder à l'expérience sur un appareil

Une fois l'expérience d'application mobile publiée, la clé d'accès est nécessaire pour accéder à l'expérience sur un appareil mobile.

Pour accéder à l'expérience sur un appareil mobile, procédez comme suit.

  1. Si vous ne disposez pas de la clé d'accès, la clé d'accès s'affiche dans le cadre des propriétés d'expérience sur le Projets page d'accueil. Cliquez sur l'expérience pour la sélectionner. La clé d'accès s'affiche dans la section des propriétés.
  2. Ouvrez l'application Brightcove Engage sur votre appareil. Entrer le Clé d'accès puis cliquez sur Prochain.
