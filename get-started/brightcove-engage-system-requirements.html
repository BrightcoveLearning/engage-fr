Configuration système requise pour Brightcove Engage

Cette page résume les exigences logicielles et matérielles requises pour créer des expériences avec Brightcove Engage, ainsi que les exigences requises pour les joueurs Brightcove.

Pour travailler avec Brightcove Engage, votre ordinateur doit répondre aux exigences suivantes.

Remarques

[1] Les anciennes versions de navigateur peuvent toujours fonctionner. Brightcove ne teste pas et ne vérifie pas la compatibilité totale avec toutes les versions du navigateur.
