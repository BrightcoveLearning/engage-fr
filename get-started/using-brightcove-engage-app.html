Utilisation de Brightcove Engage™ Application

Dans cette rubrique, vous apprendrez comment installer et utiliser l'application Brightcove Engage sur un appareil mobile.

La publication d'une expérience d'application mobile génère une clé d'accès requise par l'application Brightcove Engage pour accéder à l'expérience sur un appareil mobile. Cette rubrique couvre l'installation et la navigation de base de l'application Brightcove Engage.

Installation de l'application native iOS

L'application mobile Brightcove Engage peut être installée sur des appareils iOS exécutant iOS 11 ou une version ultérieure. Suivez ces étapes pour installer l'application.

  1. Ouvrez le Magasin d'applications sur votre appareil.
  2. Recherchez l'application Brightcove Engage.
  3. Robinet Avoir pour télécharger et installer l'application.
  4. Robinet Ouvert ou ouvrez le Brightcove s'engager application sur votre appareil. Brightcove recommande d'autoriser l'application à vous envoyer des notifications. Cela permettra à l'administrateur de l'application de vous envoyer des notifications concernant le nouveau contenu publié dans l'application.
  5. Entrer le Clé d'accès pour l'expérience.
  6. Cliquez sur Next (Suivant).

Selon la façon dont l'expérience a été configurée, vous pouvez également être invité à entrer vos informations d'identification d'entreprise pour accéder à l'application.

Installation de l'application native Android

L'application mobile Brightcove Engage peut être installée sur des appareils Android exécutant Android 5.0 ou version ultérieure. Suivez ces étapes pour installer l'application.

  1. Ouvrez le Google Play Store sur votre appareil.
  2. Recherchez l'application Brightcove Engage, puis appuyez sur Installer pour télécharger et installer l'application.
  3. Robinet Ouvert ou ouvrez le Brightcove s'engager application sur votre appareil. Brightcove recommande d'autoriser l'application à vous envoyer des notifications. Cela permettra à l'administrateur de l'application de vous envoyer des notifications sur le nouveau contenu publié dans l'application.
  4. Entrez la clé d'accès pour l'expérience.
  5. Cliquez sur Next (Suivant).

Selon la façon dont l'expérience a été configurée, vous pouvez également être invité à entrer vos informations d'identification d'entreprise pour accéder à l'application.

Navigation à l'intérieur de l'application

Lorsque vous ouvrez l'application, la vidéo en vedette apparaîtra en haut et les collections restantes apparaîtront sous la vidéo en vedette.

Faites défiler la page vers le bas pour voir toutes les collections et faites défiler vers la droite pour voir les vidéos d'une collection. Cliquez sur Voir tout pour ouvrir une page pour afficher une liste de toutes les vidéos de la collection. Cliquez sur < en haut de l'application pour revenir à la page précédente. L'application utilisera le paramètre de langue de votre appareil pour définir la langue utilisée dans l'application. Les langues suivantes sont prises en charge :

  • Anglais (États-Unis)
  • Espagnol (ES)
  • Français (FR)
  • Japonais (JP)
  • Coréen (KO)
  • Chinois (ZH)

Pendant la lecture vidéo, la rotation de votre appareil en mode paysage permet de lire la vidéo en mode plein écran.

Mise à jour de l'application

Toute modification apportée à une demande doit être republiée dans l'application Brightcove Engage. Cela peut inclure des modifications de style ou des modifications apportées aux collections de vidéos. Une fois les modifications publiées, les utilisateurs peuvent effectuer un balayage vers le bas sur l'application pour que l'application soit mise à jour avec les dernières modifications publiées.

Recherche de vidéos

L'icône de recherche () en haut de l'application peut être utilisé pour rechercher des vidéos. Seules les vidéos de l'expérience seront recherchées. Les propriétés vidéo suivantes seront recherchées :

  • Prénom
  • Short description
  • Long description
  • Balises
  • ID de la vidéo
  • Tous les champs personnalisés

Affichage de la page de profil

En cliquant sur l'icône de profil () en haut de l'application ouvrira la page Profil. La politique de confidentialité, les conditions générales et l'e-mail de contact d'aide apparaîtront sur la page Profil s'ils ont été saisis dans le cadre du détails de l'expérience.

Modification de la clé d'accès

La page Profil affiche également le nom d'utilisateur et le courriel utilisés pour se connecter à l'application si l'authentification SSO est configurée.

Pour modifier la clé d'accès, cliquez sur Se déconnecter.

Différences de joueur dans l'application Web et Engagez

Les différences de joueur suivantes seront observées dans le lecteur Web par rapport au lecteur utilisé dans l'application Engagement.

  • Les couleurs du lecteur Web ne sont pas équivalentes à celles des appareils mobiles.
  • Il y a des indicateurs de chargement sur les applications mobiles lorsque l'application est mise en mémoire tampon ou se charge pour la première fois
  • Les modifications d'orientation sont prises en charge sur les applications mobiles
  • Le volume est contrôlé à l'aide du téléphone mobile lui-même et non à partir de la barre de contrôle du média
  • Les contrôles du taux de lecture des médias ne sont pas visibles dans le lecteur Web
