Installation du Brightcove Engage™ Demande de candidatures internes

Dans cette rubrique, vous apprendrez comment installer l'application Brightcove Engage sur un appareil mobile.

Le processus d'installation décrit dans cette rubrique doit être utilisé lorsque vous avez créé votre propre application de marque avec l'aide de Brightcove Consulting. Apple et Google peuvent ne pas approuver les applications internes uniquement dans leurs magasins d'applications, ce qui nécessitera l'installation de Brightcove Engage à partir d'un autre endroit qu'un App Store.

Avant d'installer l'application, vous devez vous assurer que les fichiers d'installation nécessaires pour iOS et Android sont hébergés à un emplacement adressable/URL.

Remarque : Si vous souhaitez installer l'application Brightcove Engage depuis l'un des App Stores, consultez ce document.

La publication d'une expérience d'application mobile génère une clé d'accès requise par l'application Brightcove Engage pour accéder à l'expérience sur les appareils mobiles. Cette rubrique couvre l'installation et la navigation de base de l'application Brightcove Engage.

Installation de l'application native iOS

L'application mobile Brightcove Engage peut être installée sur des appareils iOS exécutant iOS 11 ou une version ultérieure. Suivez ces étapes pour installer l'application.

  1. Ouvrez un navigateur et accédez à l'URL où se trouve le fichier d'installation.
  2. Vous serez invité à confirmer l'installation. Cliquez sur Installer. L'application sera installée sur votre appareil.
  3. Comme l'application n'a pas été installée à partir de l'App Store d'Apple, vous devez faire confiance à l'application manuellement. Ouvrez l'appareil Réglages.
  4. Cliquez sur Général > Gestion d'appareils (il peut également apparaître comme Gestion des profils et des appareils). Votre application doit être répertoriée en tant qu'application d'entreprise.
  5. Cliquez sur votre application puis Confiance il.
  6. Cliquez sur Confiance confirmer. L'application Brightcove Engage est maintenant prête à l'emploi.
  7. Ouvrez le Engager application sur votre appareil.
  8. Entrez la clé d'accès pour l'expérience.
  9. Cliquez sur Entrée.

Selon la façon dont l'expérience a été configurée, vous pouvez également être invité à saisir les informations d'identification de votre entreprise pour accéder à l'application.

Installation de l'application native Android

L'application mobile Brightcove Engage peut être installée sur des appareils Android exécutant Android 5.0 ou version ultérieure. Notez que les étapes pour autoriser l'installation d'applications à partir de sources inconnues varient en fonction de votre version Android. Vous devez vous assurer que votre appareil autorise les installations provenant de sources inconnues.

Procédez comme suit pour installer l'application.

  1. Ouvrez l'application Paramètres sur votre appareil.
  2. dans le Personnel section, cliquez sur Sécurité.
  3. Localiser Sources inconnues et assurez-vous que le Autoriser l'installation d'applications à partir de sources autres que le Play Store est autorisé.
  4. Ouvrez un navigateur et accédez à l'URL où se trouve le fichier d'installation. Cela téléchargera l'application sur votre appareil.
  5. Confirmez l'installation en cliquant sur INSTALLER.
  6. Une fois l'installation terminée, cliquez sur OUVERT pour ouvrir l'application.
  7. Entrez la clé d'accès pour l'expérience.
  8. Cliquez sur Entrée.

Selon la façon dont l'expérience a été configurée, vous pouvez également être invité à entrer vos informations d'identification d'entreprise pour accéder à l'application.
