Modification des propriétés vidéo à l'aide de l'édition rapide

Dans cette rubrique, vous apprendrez à utiliser la fonction de modification rapide pour modifier les propriétés de la vidéo.
 

La fonctionnalité Quick Edit permet de modifier simultanément et simplement les propriétés de plusieurs vidéos. Plusieurs vidéos peuvent être sélectionnées en cliquant sur une case à cocher puis en cliquant sur une autre case à cocher (maintenez la touche Maj enfoncée tout en cliquant sur la deuxième vidéo, et les deux vidéos et toutes celles entre elles seront sélectionnées). En cliquant Edition rapide ouvrira le panneau d'édition rapide sur le côté droit de la page Vidéos.

Le volet Quick Edit affiche un sous-ensemble des propriétés vidéo :

  • Name
  • Courte description
  • Long Description
  • Tags to Add
  • Tags to Remove
  • Related Link URL
  • Texte du lien connexe
  • Scheduling Start/End Dates
  • Champs personnalisés
Remarque : Pour une description de toutes ces propriétés, consultez le Modification des propriétés de la vidéo document.

Servez-vous du volet Quick Edit pour attribuer des valeurs. Une fois terminé, cliquez sur sauver pour enregistrer les modifications ou Annuler pour quitter.

Remarques relatives à l'utilisation de Quick Edit

  • Lorsqu'au moins une vidéo a des valeurs affectées à une propriété, Edit Mixed Value s'affiche
  • Un astérisque (*) suivant le nom de la propriété indique une propriété obligatoire
  • Pour annuler les modifications apportées à une propriété, cliquez sur Restore.
