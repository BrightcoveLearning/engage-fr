Gestion des informations de compte

Dans cette rubrique, vous apprendrez comment gérer les paramètres Informations sur le compte.

La page Informations sur le compte gère le nom du compte et l'adresse postale. Notez que pour accéder à la page Informations sur le compte, vous devez être un utilisateur disposant du rôle Administrateur.

Pour accéder à la page Informations sur le compte :

  1. Connectez-vous à Brightcove Engage™.
  2. Dans l'en-tête de navigation, cliquez sur l'icône représentant une roue dentée (), puis Informations sur le compte.

Les informations relatives au compte seront affichées.

Pour modifier les informations de votre compte, cliquez sur Modifier les informations de compte.

Modifiez les valeurs appropriées, puis cliquez sur Enregistrer. Notez que l'ID de compte ne peut pas être modifié.
